# Task Manager

## Кратко описание

Да се реализира конзолно приложение на C++ за управление на задачи (To-Do списък). Приложението трябва да работи изцяло през конзолно меню. Потребителят може да създава задачи от различен тип, да ги маркира като завършени, да ги изтрива и да преглежда всички текущи задачи. Данните се запазват между сесиите във файл.

## Видове задачи

Приложението поддържа три вида задачи:

- **SimpleTask** - обикновена задача със заглавие, описание и краен срок.
- **RecurringTask** - задача, която се повтаря през фиксиран интервал (в дни). При опит за завършване на задачата, крайният срок се измества напред с интервал, вместо задачата да се маркира като завършена.
- **ProjectTask** - задача, която съдържа списък от подзадачи. Може да бъде завършена само ако всички нейни подзадачи са завършени.

Всяка задача има уникален идентификатор, заглавие, описание, краен срок, приоритет - `LOW`, `MEDIUM` или `HIGH` и статус - `PENDING` или `COMPLETED`

## Функционалности

1. Добавяне на нова задача от трите типа (Simple, Recurring, Project)
2. Маркиране на задача като завършена
3. Изтриване на задача
4. Преглед на всички задачи, сортирани по приоритет (HIGH -> LOW)
5. Зареждане на задачите от файл при стартиране и запис при изход
